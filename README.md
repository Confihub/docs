# Confi.cash documentation

Public product documentation for [Confi.cash](https://confi.cash), built with [Mintlify](https://mintlify.com).

## Local preview

```bash
npm i -g mint
mint dev
```

The preview opens at `http://localhost:3000`.

## Checks

```bash
mint broken-links
mint validate
mint a11y
```

## Editorial scope

These docs cover testnet user workflows, the public privacy model, and high-level protocol behavior. They intentionally exclude private infrastructure, administrative surfaces, secrets, key procedures, and exploit-enabling operational detail. See `AGENTS.md` for the full content boundary.

Confi.cash is experimental, testnet-only, and not independently audited. Documentation must not imply otherwise.
